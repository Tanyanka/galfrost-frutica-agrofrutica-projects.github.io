Додаткова інформація

Класність

1 клас - це чисті, стиглі й здорові плоди найвищої якості без сторонніх домішок. Такі ягоди призначені для продажу у роздрібних мережах, постачання у сферу громадського харчування, а також для подальшої переробки і виробництва джемів, желе, тощо

2 клас — це якісні плоди, але з можливими дефектами форми, поверхні, з менш суворими вимогами щодо сторонніх домішок, тощо. Вони призначені для подальшого сортування та переробки, для виробництва концентратів соку та пюре.

Пакування

  • Поліетиленові пакети по 1-2 кг в ящиках з гофрокартону
  • Гофрокартоні ящики з поліетиленовою вкладкою по 10-12 кг
  • Паперові мішки по 25 кг
  • Біг-беги по 800 кг

Виробничій процес

З 2007 р. ми заморожуємо та переробляємо корисні й смачні ягоди у м. Жовква. Львівської області. Свіжу ягоду постачають нам наші перевірені партнери відповідно до наших суворих вимог. Плоди проходять через певні цикли переробки на нашому підприємстві — див. схему нижче. А потім запакована готова продукція відправляється нашим клієнтам у транспорті з контрольованою температурою.

Використання

Наші заморожені ягоди використовують

  • для пакування продуктів у роздріб і для створення фруктових сумішей
  • для виробництва соків, смузі, пюре
  • для приготування фруктових наповнювачів для кондитерської і молочної промисловості
  • для виробництва джемів, варення, повидла, спредів тощо
  • як напівфабрикати для сублімаційного сушіння
  • в галузі харчових добавок, фармацевтиці або косметології
